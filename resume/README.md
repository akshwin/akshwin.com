# Resume Folder

Place your resume PDF file here and name it `Akshwin_T_Resume.pdf`.

The download button in the hero section will automatically link to this file.

## File Format
- **File Name**: `Akshwin_T_Resume.pdf`
- **Format**: PDF
- **Location**: `./resume/Akshwin_T_Resume.pdf`

## How to Update
1. Replace the PDF file in this folder
2. Keep the filename as `Akshwin_T_Resume.pdf`
3. The website will automatically use the new file
